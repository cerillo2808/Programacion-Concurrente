Detailed Description

Estructura que representa los parámetros y características de una placa en la simulación térmica.

Parameters
nombreArchivoNombre del archivo asociado a la simulación
nombreJobNombre del trabajo asociado a la simulación
deltaPaso de tiempo entre cálculos de temperatura
alfaDifusividad térmica de la placa
hTamaño de cada celda en la placa
epsilonPunto de equilibrio para la simulación (tolerancia para detener la iteración)
RNúmero de filas de la placa
CNúmero de columnas de la placa

Esta estructura contiene todos los parámetros necesarios para simular el comportamiento térmico de una placa, incluyendo la configuración del archivo y los parámetros físicos (delta, alfa, h, epsilon) necesarios para los cálculos.
